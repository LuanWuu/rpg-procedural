# ⚔️ Combate

O jogo possui dois sistemas de combate à distância, cada um com seus tipos de munição e efeitos.

> Voltar para [[Mecânicas]]

---

## 🔫 Sistemas

| Sistema   | Recurso     | Munição                  |
| --------- | ----------- | ------------------------ |
| [[Tiro]]  | Munição física | [[Tipos de Munição]]  |
| [[Magia]] | Mana        | [[Tipos De Magia]]       |

## ⚡ Diferenças

- **Tiro** — Dano escala com **Ataque**, reduzido por **Defesa**. Munição é craftável via [[Fabricação]]
- **Magia** — Dano escala com **Ataque Mágico**, reduzido por **Defesa Mágica**. Consome **Mana**. Possui ciclo de vantagem elemental (1.5×)
