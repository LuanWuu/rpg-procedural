# 🌀 Tipos De Magia

Índice de todos os elementos mágicos disponíveis no jogo. Cada elemento possui efeitos de status únicos.

> Voltar para [[Magia]]

---

## 🔄 Ciclo de Vantagem Elemental (1.5× de dano)

```
Fogo 🔥 → vence → Gelo 🧊 → vence → Trovão ⚡ → vence → Vento 💨 → vence → Fogo 🔥
```

## 🔥 Elementos Ofensivos

| Elemento   | Efeito de Status              | Tipo         |
| ---------- | ----------------------------- | ------------ |
| [[Fogo]]   | 🔥 Queimação (dano ao longo do tempo) | Cíclico |
| [[Gelo]]   | 🧊 Lentidão (reduz velocidade)        | Cíclico |
| [[Trovão]] | ⚡ Paralisia (impede ações)            | Cíclico |
| [[Vento]]  | 💨 Empurrão (knockback)               | Cíclico |

## 🩸 Elementos Especiais

| Elemento    | Efeito de Status                        | Tipo      |
| ----------- | --------------------------------------- | --------- |
| [[Sangue]]  | 🩸 Vampirismo + Sangramento             | Especial  |
| [[Suporte]] | ✨ Buffs para aliados / Debuffs em inimigos | Especial |

> ⚠️ **Sangue** só funciona em inimigos com sangue. **Suporte** requer acertar o alvo (aliado ou inimigo).
