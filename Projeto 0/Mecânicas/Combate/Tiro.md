# 🔫 Tiro

Sistema de combate à distância com munição física. Diferente da [[Magia]], o tiro usa munição craftável com efeitos variados.

> Voltar para [[Mecânicas]] > [[Combate]]

---

## ⚡ Como Funciona

- Armas de tiro consomem **munição física** (não mana)
- Cada tipo de munição possui um **efeito especial** com chance de ativação
- O dano escala com **Ataque** e é reduzido por **Defesa** (ver [[Atributos]])
- Munição pode ser fabricada via [[Fabricação]] ou comprada

## 🎯 Tipos de Munição

Ver detalhes completos em [[Tipos de Munição]].

| Munição              | Efeito                          |
| -------------------- | ------------------------------- |
| **Bala Incendiária** | 🔥 Chance de causar queimação   |
| **Bala de Impacto**  | 🧊 Chance de causar lentidão    |
| **Bala Penetrante**  | 🛡️ Reduz defesa do inimigo     |
