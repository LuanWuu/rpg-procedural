# 🎮 RPG

Visão geral do **Projeto 0** — um RPG com geração procedural.

---

## 📈 Sistema de Nível

- O personagem ganha **experiência** ao derrotar inimigos e completar conteúdo
- A cada nível, recebe **10 pontos** para distribuir em [[Atributos]] (fixos e multiplicadores)
- Todos os atributos ganham **+1 ponto fixo** automaticamente por nível

## 🎯 Pilares do Gameplay

| Pilar            | Descrição                                                  |
| ---------------- | ---------------------------------------------------------- |
| **Combate**      | [[Tiro]] e [[Magia]] — dois sistemas de combate à distância |
| **Exploração**   | [[Dungeon]] — dungeons procedurais com [[Biomas]]           |
| **Crafting**     | [[Fabricação]] — criar itens via minigames                  |
| **Progressão**   | [[Atributos]] e [[Árvore de Skill]] — builds customizáveis |

## 🎨 Identidade Visual

- Estilo **3D Low Poly**
- Estética **Dieselpunk Pós-Apocalíptica**
- Inimigos com modelos **procedurais** (ver [[Inimigos]])
