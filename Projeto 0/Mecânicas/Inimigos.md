# 👹 Inimigos

Sistema de inimigos com modelos gerados **proceduralmente**.

> Voltar para [[Mecânicas]] | Visual: [[Arte]]

---

## 🎲 Geração Procedural

- Modelos de inimigos são gerados proceduralmente, criando variedade visual
- Cada bioma da [[Dungeon]] possui inimigos temáticos
- Minibosses e bosses são versões mais fortes dos inimigos do bioma

## 🩸 Tipos de Inimigos

| Tipo                  | Vulnerabilidade a [[Sangue]]  |
| --------------------- | ----------------------------- |
| **Criaturas vivas**   | ✅ Vulnerável (tem sangue)    |
| **Construtos/Elementais** | ❌ Imune (sem sangue)     |

> 💡 Escolher o elemento certo contra cada tipo de inimigo é parte da estratégia (ver [[Tipos De Magia]]).
