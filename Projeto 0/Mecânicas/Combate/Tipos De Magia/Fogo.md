# 🔥 Fogo

Elemento ofensivo do ciclo elemental. Causa **dano ao longo do tempo**.

> Voltar para [[Tipos De Magia]]

---

## ⚡ Efeito de Status

- **Queimação** — Chance de aplicar dano contínuo por alguns segundos
- Dano escala com **Ataque Mágico** (ver [[Atributos]])

## 🔄 Vantagem Elemental

- 🟢 **Forte contra:** [[Gelo]] (1.5× dano)
- 🔴 **Fraco contra:** [[Vento]] (recebe 1.5× dano)
