# 🔮 Magia

Sistema de combate mágico. Armas mágicas utilizam **mana como munição**, com diferentes elementos que causam efeitos de status variados.

> Voltar para [[Mecânicas]] > [[Combate]]

---

## ⚡ Como Funciona

- Armas mágicas consomem **mana** ao disparar (ver [[Atributos]])
- Cada elemento possui um **efeito de status** com chance de ativação
- Existe um **ciclo de vantagem elemental** com buff de 1.5× de dano
- O dano escala com **Ataque Mágico** e é reduzido por **Defesa Mágica**

## 🔄 Ciclo de Vantagem Elemental

```
Fogo 🔥 → Gelo 🧊 → Trovão ⚡ → Vento 💨 → Fogo 🔥
                    (buff de 1.5×)
```

> Sangue e Suporte ficam **fora do ciclo** — têm mecânicas próprias.
