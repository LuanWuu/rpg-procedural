# 🔨 Fabricação

Sistema de crafting do jogo. O jogador fabrica itens manualmente através de minigames, com opção de delegar para NPCs.

> Voltar para [[Mecânicas]]

---

## 🛠️ Disciplinas

| Disciplina      | Descrição                                      |
| --------------- | ---------------------------------------------- |
| **Forja**       | Criação de armas e armaduras metálicas          |
| **Alquimia**    | Criação de poções e itens consumíveis           |
| **Manufatura**  | Criação de equipamentos, acessórios e materiais |
| **Cozinha**     | Preparação de comidas com buffs temporários     |

## 🎮 Gameplay

- Cada disciplina funciona como um **minigame** manual
- O jogador pode **pular o processo** pagando um NPC para fabricar
- A qualidade do item depende da **proficiência** do jogador na disciplina

## 📈 Proficiência (FAMC)

- Cada disciplina tem sua própria proficiência: **F**orja, **A**lquimia, **M**anufatura, **C**ozinha
- Proficiência **não sobe com nível** — sobe apenas por:
  - Produtos fabricados com sucesso
  - Completar **sidequests** (missões secundárias) relacionadas
