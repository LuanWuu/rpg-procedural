# 🧊 Gelo

Elemento ofensivo do ciclo elemental. Causa **redução de velocidade**.

> Voltar para [[Tipos De Magia]]

---

## ⚡ Efeito de Status

- **Lentidão** — Chance de reduzir a velocidade de movimento do inimigo
- Dano escala com **Ataque Mágico** (ver [[Atributos]])

## 🔄 Vantagem Elemental

- 🟢 **Forte contra:** [[Trovão]] (1.5× dano)
- 🔴 **Fraco contra:** [[Fogo]] (recebe 1.5× dano)
