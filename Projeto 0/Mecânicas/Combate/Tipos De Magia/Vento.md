# 💨 Vento

Elemento ofensivo do ciclo elemental. Causa **knockback** (empurrão).

> Voltar para [[Tipos De Magia]]

---

## ⚡ Efeito de Status

- **Empurrão** — Chance de empurrar inimigos para trás (knockback)
- Útil para controle de posicionamento em [[Dungeon]]
- Dano escala com **Ataque Mágico** (ver [[Atributos]])

## 🔄 Vantagem Elemental

- 🟢 **Forte contra:** [[Fogo]] (1.5× dano)
- 🔴 **Fraco contra:** [[Trovão]] (recebe 1.5× dano)
