# ⚡ Trovão

Elemento ofensivo do ciclo elemental. Causa **paralisia**.

> Voltar para [[Tipos De Magia]]

---

## ⚡ Efeito de Status

- **Paralisia** — Chance de impedir o inimigo de agir por alguns segundos
- Dano escala com **Ataque Mágico** (ver [[Atributos]])

## 🔄 Vantagem Elemental

- 🟢 **Forte contra:** [[Vento]] (1.5× dano)
- 🔴 **Fraco contra:** [[Gelo]] (recebe 1.5× dano)
